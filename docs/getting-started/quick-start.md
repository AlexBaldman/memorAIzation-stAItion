By the end of this guide, you'll be able to:

- Navigate the memory training interface

## 🎯 **What This Is**

**memorAIzation stAItion** is a cutting-edge memory training platform that combines:

- **🎭 PAO System**: Maps numbers 00-99 to memorable celebrity associations
- **🎯 Peg System**: Visual anchors for digits 0-9 using intuitive imagery
- **🎲 Dice Practice**: Interactive training for generating and practicing PAO numbers
- **🎨 AI-Powered Imagery**: Dynamic image generation to reinforce memory associations

## ⚡ **Quick Setup**

### **1. Clone and Install**

```bash
git clone <repository-url>
cd memorAIzation-stAItion
npm install
```

### **2. Start Development Server**

```bash
npm run dev
```

### **3. Open Your Browser**

Navigate to `http://localhost:5173` (or the URL shown in your terminal)

## 🧠 **Your First Memory Training Session**

### **Step 1: Explore the Interface**

The main interface consists of:

- **Header**: AI configuration and accessibility controls
- **Card Grid**: Memory cards for numbers 00-99
- **Dice Practice**: Interactive dice rolling section
- **Peg System**: Visual anchors for digits 0-9
- **Performance Dashboard**: System insights and metrics

### **Step 2: Try the PAO System**

1. **Look at a card** (e.g., card 00 - Ozzy Osbourne)
2. **Click to flip** and see the full description
3. **Practice recalling**: What number is Ozzy Osbourne? (Answer: 00)
4. **Reverse practice**: What celebrity is 00? (Answer: Ozzy Osbourne)

### **Step 3: Dice Practice**

1. **Roll the dice** to generate a random number
2. **Practice recalling** the celebrity for that number
3. **Check your accuracy** in the session history
4. **Monitor your progress** with performance statistics

### **Step 4: AI Image Generation**

1. **Flip a card** to see the back
2. **Click "Generate Image"** to create a new AI image
3. **Try "Edit Image"** to modify existing images with prompts
4. **Watch the magic** as AI creates personalized memory aids

## 🎯 **Memory Training Tips**

### **PAO System Mastery**

- **Start with familiar numbers**: Practice 00-09 first
- **Create vivid mental images**: The more bizarre, the better
- **Use the emoji stories**: Visual cues enhance memory
- **Practice regularly**: Consistency beats intensity

### **Dice Practice Strategy**

- **Set daily goals**: Aim for 20-30 rolls per session
- **Track your progress**: Monitor accuracy over time
- **Mix it up**: Practice both number-to-celebrity and reverse
- **Use the statistics**: Identify areas for improvement

### **AI Image Enhancement**

- **Be specific**: Detailed prompts create better images
- **Experiment with styles**: Try different artistic approaches
- **Use for difficult numbers**: Generate images for hard-to-remember associations
- **Cache management**: Save your best generated images
