# Security Policy

## Supported Versions

We are currently supporting the latest version of the application on the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

The team and community take all security bugs in this project seriously. Thank you for improving the security of our project. We appreciate your efforts and responsible disclosure and will make every effort to acknowledge your contributions.

To report a security vulnerability, please email `security@example.com` and include the word "SECURITY" in the subject line.

We will endeavor to respond to your report within 48 hours, and will keep you informed of the progress of our investigation and resolution. We will not disclose your identity without your permission.

Please do not report security vulnerabilities through public GitHub issues.
